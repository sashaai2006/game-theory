# Стартап vs Рынок

Интерактивное приложение на **Streamlit** для разбора антагонистической матричной игры с нулевой суммой: стратегии «стартапа» против состояний «рынка». В качестве платежей используются значения рентабельности (в процентах); игрок A максимизирует выигрыш, игрок B минимизирует его.

## Возможности

- **Платёжная матрица** задаётся в `config.py` и подхватывается расчётным ядром без правок UI.
- **Доминирование стратегий**, проверка седловой точки, редуцирование матрицы.
- **Три способа найти решение**: аналитика для блока 2×2 после вычёркивания, линейное программирование (SciPy HiGHS), итерации Брауна–Робинсона.
- **Графики Plotly**: нижняя огибающая, сходимость оценок цены игры, интерактивный сценарий со слайдерами.

Структура проекта:

| Файл | Назначение |
|------|------------|
| `app.py` | Интерфейс Streamlit, визуализация и пошаговое объяснение |
| `matrix_game.py` | Расчёты: седло, доминирование, ЛП, Браун–Робинсон |
| `config.py` | Имена строк/столбцов и числовая матрица |

## Требования

- Python **3.11+**

Зависимости описаны в **`pyproject.toml`** (раздел `[project]` и опционально `[project.optional-dependencies]` для разработки).

## Установка

Рекомендуется виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```

Только зависимости без установки пакета:

```bash
pip install streamlit numpy scipy pandas plotly
```

Инструменты для проверки кода (опционально):

```bash
pip install -e ".[dev]"
```

## Запуск

```bash
streamlit run app.py
```

Откройте адрес, который покажет Streamlit в терминале (обычно `http://localhost:8501`).

## Идея задачи

После исключения доминируемых столбцов игра сводится к матрице 2×2; смешанное равновесие Нэша и цена игры согласуются между аналитической формулой, решением ЛП и методом Брауна–Робинсона. Во вкладке «Математика шаг за шагом» приведены постановка, проверки и сравнение методов.

---

Учебный проект по теории игр; при изменении сценария достаточно обновить матрицу и подписи в `config.py`.
